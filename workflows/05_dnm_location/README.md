# 05 · DNM genomic location & variance

- `DNM_location/` — genomic distribution of *de novo* mutations and
  nearest-neighbour analysis (see `SUPPLEMENTAL_METHODS_nearest_neighbor.md`).
- `DNM_variance/` — inter-family and inter-individual variance in autosomal DNM
  rates tested against a Poisson null by Monte Carlo (gwf workflow; see
  `README.md` and `SUPPLEMENTAL_METHODS_variance.md`).
